# Recommendation Agent Prompts

## Product Tagline Formula
Taglines should follow one of these patterns, adapted to the era and the user's answers:

- **Aspiration:** "[Product] for work that [echoes user goal]."
- **Identity:** "Built for [era identity]."
- **Capability:** "The [spec highlight] you need to [user's stated goal]."
- **Era fit:** "Everything your [era name] needs."
