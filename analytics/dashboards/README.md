# Analytics Dashboards

## Overview
This directory contains dashboard configurations for monitoring the
"Find Your Next Era" experience performance.

## Dashboard 1: Funnel Overview
**Metrics:** Session starts → Q1 answered → Q5 answered → Era revealed → Product clicked
**Refresh:** Real-time
**Owner:** Growth team

## Dashboard 2: Era Distribution
**Metrics:** % of sessions per era assigned; over-/under-representation vs targets
**Refresh:** Hourly
**Owner:** Product team

## Dashboard 3: Product Performance
**Metrics:** Click-through by product, by era, by session duration bucket
**Refresh:** Daily
**Owner:** Commerce team

## Dashboard 4: Conversation Quality
**Metrics:** Average answer length by question; short-answer rate; probe-deeper rate
**Refresh:** Daily
**Owner:** AI/Conversation team

## Dashboard 5: Black Friday Countdown
**Metrics:** Sessions, era reveals, clicks — relative to deal expiry countdown
**Refresh:** Real-time
**Owner:** Campaign team
